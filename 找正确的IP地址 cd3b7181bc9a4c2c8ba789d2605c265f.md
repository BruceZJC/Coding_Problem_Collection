# 找正确的IP地址

知识点Key: IP地址正则表达式，具体题会稍微有不一样
解题方法Methods: 正则表达式RegExp
难度Level: 简单Easy

String pattern = "^((0|00[0-9]|0[1-9][0-9]|[1-9][0-9]|1[0-9][0-9]|2[0-4][0-9]|25[0-5])(\\.(?!$)|$)){4}$";

分析：

1. ^( ( 0 | 00[0-9] | 0[1-9][0-9] | [1-9][0-9] | 1[0-9][0-9] | 2[0-4][0-9] | 25[0-5])
2. (\\.(?!$)|$) ){4}$
